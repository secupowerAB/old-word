### 第三部分：核心功能演示（90秒）

#### 3.1 输入主题（20秒）
**画面**：

- 输入"春江花月夜"
- 展示输入框的动画效果

**解说词**：
"使用非常简单，只需在输入框中输入您想要的诗词主题。系统提供了快速选择按钮，也可以自由输入任何主题。"

#### 3.2 生成诗词（40秒）
**画面**：
1. 点击"挥毫泼墨"按钮
2. 显示毛笔书写的加载动画
3. 等待AI生成
4. 诗词逐字显示在画卷上

**解说词**：
"点击'挥毫泼墨'按钮，系统会调用DeepSeek AI模型开始创作。优美的加载动画模拟了毛笔书写的过程。几秒钟后，一首符合格律的古诗就生成了，诗词会逐字显示，就像在宣纸上缓缓晕开。"

#### 3.3 操作功能（30秒）
**画面**：
1. 点击"复制"按钮
2. 打开记事本，粘贴诗词
3. 返回页面，点击"重新生成"
4. 显示新的诗词

**解说词**：
"生成的诗词可以一键复制到剪贴板，方便保存和分享。如果对当前诗词不满意，可以点击'重新生成'按钮，系统会用相同主题创作新的诗词。"

## 录制建议

### 工具推荐
- **屏幕录制**：
  - OBS Studio（免费）
  - Camtasia（专业）
  - QuickTime（Mac）
  - Xbox Game Bar（Windows）

- **视频编辑**：
  - DaVinci Resolve（免费）
  - Adobe Premiere Pro
  - iMovie（Mac）
  - 剪映

### 录制技巧

1. **前期准备**
   - 清理桌面，关闭无关程序
   - 准备好演示内容
   - 测试录制软件和麦克风
   - 调整浏览器窗口大小（推荐1920x1080）

2. **录制过程**
   - 保持鼠标移动流畅
   - 突出重点操作
   - 控制录制节奏
   - 避免过长的等待时间

3. **后期制作**
   - 剪辑掉失误片段
   - 添加背景音乐（可选）
   - 添加字幕说明
   - 添加转场效果
   - 调整音量和画质

### 画质建议
- 分辨率：1920x1080（Full HD）
- 帧率：30fps或60fps
- 比特率：5-8 Mbps
- 格式：H.264 / MP4

### 音频建议
- 清晰的人声
- 无背景噪音
- 适中的音量
- 可选择添加轻音乐

```

```
