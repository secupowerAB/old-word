### 1.4 软件简介
墨韵诗生是一款基于AI技术的中国古诗词生成Web应用。用户只需输入一句话或主题，系统即可通过DeepSeek AI模型生成符合格律的优美古诗词。软件采用中国传统水墨画美学设计，融合现代Web技术，为用户提供优雅的诗词创作体验。

## 二、功能特性

### 2.1 核心功能

#### 2.1.1 智能诗词生成
- 输入主题或一句话，自动生成古诗词

- 生成的诗词符合古典诗词的格律和韵律
- 意境优美，语言典雅

#### 2.1.2 快速主题选择
- 预设常用诗词主题（春日江南烟雨、明月思乡、山水之间、秋风落叶等）
- 一键填入，快速生成

#### 2.1.3 诗词展示
- 采用竹简卷轴样式展示诗词
- 诗词逐字显示，营造书写感
- 优雅的动画效果

#### 2.1.4 诗词管理
- 一键复制诗词到剪贴板

### 3.3 数据流程

1. 用户在输入框输入主题或选择预设主题
2. 点击"挥毫泼墨"按钮触发生成事件
3. JavaScript验证输入内容
4. 构建系统提示词和用户提示词
5. 通过Fetch API向DeepSeek发送POST请求
6. 接收AI生成的诗词内容
7. 解析JSON响应数据
8. 格式化诗词文本
9. 在页面上逐字显示诗词
10. 显示操作按钮（复制、重新生成）

## 四、安装与配置

### 4.1 系统要求

#### 4.1.1 运行环境
- 现代Web浏览器（Chrome 90+、Firefox 88+、Safari 14+、Edge 90+）
- 支持JavaScript
- 稳定的网络连接

#### 4.1.2 开发环境（可选）
- 文本编辑器（VS Code、Sublime Text等）
- HTTP服务器（Python、Node.js等）
- Git版本控制

### 4.2 安装步骤

#### 4.2.1 下载项目
```bash
# 方式1：Git克隆
git clone https://github.com/your-username/poetry-generator.git
cd poetry-generator

# 方式2：下载ZIP
# 从GitHub下载ZIP文件并解压
```

#### 4.2.2 配置API密钥
1. 访问DeepSeek平台：https://platform.deepseek.com/
2. 注册账号并登录
3. 创建API密钥
4. 复制config-example.js为config.js
5. 将API密钥填入config.js

```javascript
const DEEPSEEK_API_KEY = 'sk-xxxxxxxxxxxxxx'; // 替换为实际密钥
```

#### 4.2.3 启动服务

**使用Python**
```bash
python -m http.server 8000
# 访问 http://localhost:8000
```

**使用Node.js**
```bash
npm install -g http-server
http-server
# 访问 http://localhost:8080
```

**直接打开**
- 双击index.html文件
- 在浏览器中打开

### 4.3 配置说明

#### 4.3.1 API配置
在config.js中可以调整以下参数：

```javascript
const CONFIG = {
    apiBaseUrl: 'https://api.deepseek.com/v1',
    modelName: 'deepseek-chat',
    temperature: 0.8,      // 创意程度 (0-2)
    maxTokens: 500,        // 最大生成长度
    timeout: 30000         // 请求超时时间（毫秒）
};
```

#### 4.3.2 温度参数调整
- 0.3-0.5：生成更规范、严谨的诗词
- 0.6-0.8：平衡创意和规范（推荐）
- 0.9-1.2：更有创意，但可能偏离传统格律

## 五、使用说明

### 5.1 基本操作

#### 5.1.1 生成诗词
1. 在输入框中输入诗词主题或一句话
2. 或点击快速主题按钮
3. 点击"挥毫泼墨"按钮
4. 等待AI生成诗词（约2-5秒）
5. 查看生成的诗词

#### 5.1.2 复制诗词
1. 生成诗词后，点击"复制"按钮
2. 诗词将被复制到剪贴板
3. 可粘贴到任何文本编辑器

#### 5.1.3 重新生成
1. 点击"重新生成"按钮
2. 系统将使用相同主题生成新的诗词
3. 每次生成的诗词都可能不同

### 5.2 最佳实践

#### 5.2.1 主题选择建议
- 使用简洁的词语或短语（3-10字为宜）
- 可以是景物、情感、季节、事件等
- 避免过于复杂或抽象的描述

#### 5.2.2 示例主题
- 景物类：春江花月夜、山水之间、江南烟雨
- 情感类：思念故乡、离别之情、相思之苦
- 季节类：秋风落叶、冬雪梅花、夏日荷塘
- 事件类：登高望远、泛舟江上、夜读诗书

### 5.3 常见问题

#### 5.3.1 API调用失败
- 检查网络连接
- 确认API密钥配置正确
- 检查API额度是否充足
- 查看浏览器控制台错误信息

#### 5.3.2 生成诗词不理想
- 尝试调整主题描述
- 点击"重新生成"换一首
- 调整config.js中的temperature参数

#### 5.3.3 页面显示异常
- 刷新页面
- 清除浏览器缓存
- 检查浏览器版本
- 使用推荐的现代浏览器

## 八、安全性说明

### 8.1 数据安全

1. **API密钥保护**
   - 密钥存储在本地config.js
   - .gitignore防止密钥泄露
   - 建议定期更换密钥

2. **数据传输**
   - 使用HTTPS加密传输
   - 不存储用户输入数据
   - 不收集用户个人信息

3. **隐私保护**
   - 无用户追踪
   - 无数据统计
   - 无第三方分析工具

### 8.2 安全建议

1. 不要将包含真实API密钥的代码上传到公开仓库
2. 定期检查API使用情况
3. 设置API调用频率限制
4. 保持软件更新
